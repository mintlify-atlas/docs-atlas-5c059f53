---
title: "MSA sampling utilities"
description: "Sample homologous sequences from an MSA for retrieval-augmented inference."
---

The `E1.msa_sampling` module provides utilities for building retrieval-augmented context from a multiple-sequence alignment (MSA) file. The sampled context sequences are passed to `E1Predictor.predict` or `E1Scorer.score` via the `context_seqs` argument.

MSA files must be in **A3M format** with the query sequence as the first record. Lowercase characters (insertions relative to the query) are ignored automatically.

---

## `ContextSpecification`

Dataclass that bundles sampling hyperparameters for a single context draw.

```python
@dataclass
class ContextSpecification:
    max_num_samples: int = 511
    max_token_length: int = 32768
    max_query_similarity: float = 1.0
    min_query_similarity: float = 0.0
    neighbor_similarity_lower_bound: float = 0.8
```

<ParamField path="max_num_samples" type="integer" default="511">
  Upper bound on the number of sequences included in the context. The actual count may be smaller if `max_token_length` is reached first.
</ParamField>

<ParamField path="max_token_length" type="integer" default="32768">
  Maximum total residue count (after gap removal) across all context sequences. Sequences are added greedily in sample order until this limit would be exceeded.
</ParamField>

<ParamField path="max_query_similarity" type="float" default="1.0">
  Sequences with similarity to the query above this threshold are excluded before sampling. Set to a value below `1.0` to exclude near-identical homologs.
</ParamField>

<ParamField path="min_query_similarity" type="float" default="0.0">
  Sequences with similarity to the query below this threshold are excluded before sampling.
</ParamField>

<ParamField path="neighbor_similarity_lower_bound" type="float" default="0.8">
  Similarity threshold used to compute per-sequence neighbor counts. Each sequence is sampled with weight proportional to `1 / num_neighbors`, where a neighbor is any MSA sequence with similarity ≥ this bound. This down-weights over-represented sequence clusters.
</ParamField>

## `sample_context`

```python
def sample_context(
    msa_path: str,
    max_num_samples: int,
    max_token_length: int,
    max_query_similarity: float = 1.0,
    min_query_similarity: float = 0.0,
    neighbor_similarity_lower_bound: float = 0.8,
    use_full_sequences_in_context: bool = False,
    full_sequences_path: str | None = None,
    seed: int = 0,
    device: torch.device | None = None,
    cache_num_neighbors_path: str | None = None,
) -> tuple[str, list[str]]
```

Samples a single context from an MSA file.

<ParamField path="msa_path" type="string" required>
  Path to the A3M-format MSA file. The first sequence is treated as the query.
</ParamField>

<ParamField path="max_num_samples" type="integer" required>
  Maximum number of homologs to draw. See `ContextSpecification.max_num_samples`.
</ParamField>

<ParamField path="max_token_length" type="integer" required>
  Maximum total residue count for the context. See `ContextSpecification.max_token_length`.
</ParamField>

<ParamField path="max_query_similarity" type="float" default="1.0">
  Exclude sequences more similar to the query than this value.
</ParamField>

<ParamField path="min_query_similarity" type="float" default="0.0">
  Exclude sequences less similar to the query than this value.
</ParamField>

<ParamField path="neighbor_similarity_lower_bound" type="float" default="0.8">
  Similarity threshold for neighbor-count computation used in inverse-frequency sampling weights.
</ParamField>

<ParamField path="use_full_sequences_in_context" type="boolean" default="false">
  When `True`, full (unaligned) sequences from `full_sequences_path` are used as context instead of the aligned MSA fragments.
</ParamField>

<ParamField path="full_sequences_path" type="string" default="None">
  Path to a FASTA file of full-length sequences. Required when `use_full_sequences_in_context=True`. Must contain the same number of records in the same order as `msa_path`, with matching IDs.
</ParamField>

<ParamField path="seed" type="integer" default="0">
  Random seed for reproducible sampling.
</ParamField>

<ParamField path="device" type="torch.device" default="None">
  Device for byte-tensor similarity computations. Defaults to CUDA if available, otherwise CPU.
</ParamField>

<ParamField path="cache_num_neighbors_path" type="string" default="None">
  Path to a `.npy` file for caching neighbor counts. On first call the counts are computed and saved; on subsequent calls the file is loaded, avoiding recomputation for large MSAs.
</ParamField>

**Returns** `tuple[str, list[str]]`:

<ResponseField name="context_string" type="string">
  Comma-separated string of sampled context sequences, ready for use as a key in `context_seqs` dicts or as part of a multi-sequence input string.
</ResponseField>

<ResponseField name="context_ids" type="list[string]">
  FASTA record IDs of the sampled sequences, in the order they appear in the context string.
</ResponseField>

## `sample_multiple_contexts`

```python
def sample_multiple_contexts(
    msa_path: str,
    context_specifications: list[ContextSpecification],
    use_full_sequences_in_context: bool = False,
    full_sequences_path: str | None = None,
    seed: int = 0,
    device: torch.device | None = None,
    cache_num_neighbors_path: str | None = None,
) -> tuple[list[str], list[list[str]]]
```

Draws multiple independent context samples from the same MSA, each governed by its own `ContextSpecification`. Useful for ensembling over diverse context sets.

Neighbor counts are computed once and shared across all context draws via an internal temporary cache (or `cache_num_neighbors_path` when supplied).

<ParamField path="msa_path" type="string" required>
  Path to the A3M MSA file.
</ParamField>

<ParamField path="context_specifications" type="ContextSpecification[]" required>
  List of `ContextSpecification` instances. One context is sampled per entry. Each draw uses `seed + i` for reproducibility.
</ParamField>

<ParamField path="use_full_sequences_in_context" type="boolean" default="false">
  Use full unaligned sequences for all contexts. See `sample_context`.
</ParamField>

<ParamField path="full_sequences_path" type="string" default="None">
  Required when `use_full_sequences_in_context=True`.
</ParamField>

<ParamField path="seed" type="integer" default="0">
  Base random seed. Context `i` uses `seed + i`.
</ParamField>

<ParamField path="device" type="torch.device" default="None">
  Computation device. Defaults to CUDA if available.
</ParamField>

<ParamField path="cache_num_neighbors_path" type="string" default="None">
  Path for neighbor-count cache. A temporary file is used when `None`.
</ParamField>

**Returns** `tuple[list[str], list[list[str]]]`:

<ResponseField name="contexts" type="list[string]">
  List of comma-separated context strings, one per `ContextSpecification`.
</ResponseField>

<ResponseField name="context_ids" type="list[list[string]]">
  List of ID lists, one inner list per context.
</ResponseField>

## Usage example

```python
import torch
from E1.msa_sampling import ContextSpecification, sample_context, sample_multiple_contexts
from E1.modeling import E1ForMaskedLM
from E1.predictor import E1Predictor

# --- Single context ---
context_str, context_ids = sample_context(
    msa_path="query.a3m",
    max_num_samples=511,
    max_token_length=32768,
    max_query_similarity=0.9,   # exclude sequences >90% identical to query
    seed=42,
    cache_num_neighbors_path="num_neighbors.npy",
)
print(f"Sampled {len(context_ids)} sequences, total length {len(context_str.replace(',', ''))}")

# --- Multiple contexts for ensembling ---
specs = [
    ContextSpecification(max_num_samples=127, max_token_length=8192),
    ContextSpecification(max_num_samples=255, max_token_length=16384),
    ContextSpecification(max_num_samples=511, max_token_length=32768),
]
contexts, all_ids = sample_multiple_contexts("query.a3m", context_specifications=specs, seed=0)

# Pass multiple contexts to the predictor
model = E1ForMaskedLM.from_pretrained("Profluent-Bio/E1-300m").to("cuda:0")
predictor = E1Predictor(model)

context_seqs = {f"ctx_{i}": ctx for i, ctx in enumerate(contexts)}
for pred in predictor.predict(["MKTIIALSYIFCLVFA"], context_seqs=context_seqs):
    print(pred["id"], pred["context_id"], pred["mean_token_embeddings"].shape)
```
