Profluent E1 is a family of protein language models (150M–600M parameters) that generate high-quality sequence embeddings and per-residue logits. E1 models are drop-in replacements for the ESM model family, supporting both single-sequence and retrieval-augmented (homolog-aware) inference out of the box.

## Available models

| Model | Parameters | HuggingFace |
|-------|-----------|-------------|
| E1-150m | 150M | [Profluent-Bio/E1-150m](https://huggingface.co/Profluent-Bio/E1-150m) |
| E1-300m | 300M | [Profluent-Bio/E1-300m](https://huggingface.co/Profluent-Bio/E1-300m) |
| E1-600m | 600M | [Profluent-Bio/E1-600m](https://huggingface.co/Profluent-Bio/E1-600m) |

Model weights are downloaded automatically from HuggingFace on first use.

## Get started

<Steps>
  <Step title="Install the package">
    Clone the repository and install with pip:
    ```bash
    git clone https://github.com/Profluent-AI/E1.git
    cd E1 && pip3 install -e .
    ```
  </Step>
  <Step title="Load a model">
    Models are loaded directly from HuggingFace:
    ```python
    from E1.modeling import E1ForMaskedLM
    model = E1ForMaskedLM.from_pretrained("Profluent-Bio/E1-300m").to("cuda:0")
    model.eval()
    ```
  </Step>
  <Step title="Run inference">
    Prepare a batch and run a forward pass:
    ```python
    from E1.batch_preparer import E1BatchPreparer
    batch_preparer = E1BatchPreparer()
    batch = batch_preparer.get_batch_kwargs(["MFCATEEKL"], device="cuda:0")
    with torch.autocast("cuda", dtype=torch.bfloat16, enabled=True):
        outputs = model(
            input_ids=batch["input_ids"],
            within_seq_position_ids=batch["within_seq_position_ids"],
            global_position_ids=batch["global_position_ids"],
            sequence_ids=batch["sequence_ids"],
            past_key_values=None,
            use_cache=False,
        )
    embeddings = outputs.embeddings  # (B, L, E)
    ```
  </Step>
</Steps>

<Tip>
  For GPU users with CUDA capability 8.0+ (Ampere and newer), install `flash-attn` for faster inference:
  ```bash
  pip3 install flash-attn --no-build-isolation
  ```
</Tip>
